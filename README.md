# Face-recognition
Key Features 
1. Real-time face recognition through live camera 
2. Works with existing CCTV or webcam setups 
3. Automatic attendance tracking (IN / OUT capture) 
4. Multi-person detection and recognition 
5. Can be used for employee monitoring and theft prevention 
5. Attendance data logging for analysis and reporting 
⚡ Contactless and automated system
